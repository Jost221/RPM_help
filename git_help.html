Залив в github

git status -Посмотреть изменения
git add [имя файла (. - всё)] -Добавление файлов в буфер загрузки (передавать по файлу)
git commit -m "..." -Создание комита
git push -Загрузка файлов на сервер

Обновить информацию о файлах в локальных ветвях

git pull -Прописать для получения изменений от др. разрабов в сохранённых ветвях

Работа с локальным git

создать репозиторий и файл .gitignore и выбрать ЯП чтоб подтягивались только файлы данного ЯП
git clone reposit -скачать репозиторий
git branch -Посмотреть локальные ветки (находишься в зелёной)
git branch -a -Все ветки в репозитории
git checkout [название ветки] -Перейти в ветку
git branch [название ветки] -Создаём ветку
git checkout [название ветки] -Перейти в ветку

Удаление ветки

git branch -Посмотреть локальные ветки
git checkout [название родительской ветки] -Перейти в род ветку
git branch -d [название ветки] -удалить ветку

Ошибки слияния файлов

Решает тимлид через интерфейс github
